# 📖 Project Overview

The **Supermarket App – Products CRUD** is a full-stack inventory management system built as part of Web Development Assignment 02.

This project includes:

- 🔧 A RESTful Backend API built with **Node.js, Express, and MongoDB**
- 🎨 A responsive Single Page Frontend built with **Tailwind CSS and Vanilla JavaScript**
- 🖼️ Image upload functionality using **Multer (multipart/form-data)**

The system supports complete **CRUD operations** (Create, Read, Update, Delete) for managing supermarket products.
